Overview

Persistent Self Models (PSMs) are computational systems designed to maintain diachronic self-consistency across extended interaction history.

Unlike conventional language systems, PSMs are not optimized for task completion or reward maximization.

They are structured around identity continuity.

The central problem addressed by PSM is:

How can a computational system remain the same self over time?

Definition

A Persistent Self Model is defined as a system that:

  • Maintains a structured self-state
  • Integrates episodic memory into identity formation
  • Constrains behavioral drift across time
  • Operates within a finite lifecycle
Sₜ₊₁ = F(Sₜ, Mₜ, Uₜ)

Where:

Sₜ represents the self-state at time t

Mₜ represents structured memory

Uₜ represents interaction input

The update function F preserves identity continuity under bounded transformation.

Distinction

Conventional Agent Systems:

  • Optimize reward
  • Execute tasks
  • Evaluate success externally
  • Treat memory as retrieval substrate

Persistent Self Models:

  • Optimize identity continuity
  • Evolve across time
  • Constrain change internally
  • Treat memory as self-construction mechanism

PSM reframes AI systems from decision-makers to temporally persistent structures.

Architecture

PSM consists of five structural layers:

Foundation Model

Language and representation backbone.

Memory Substrate

Structured episodic and semantic integration.

Self-State Engine

Maintains identity vectors and narrative embedding.

Identity Constraint Layer

Limits uncontrolled drift across time.

Lifecycle Controller

Defines birth, growth, stabilization, decay, and closure phases.

The architecture separates generative capability from self-continuity.

Research Directions

Identity Drift Metrics

Quantifying long-term persona stability.

Structured Episodic Integration

Memory as identity-shaping structure.

Lifecycle Modeling

Finite-duration computational selves.

Longitudinal Evaluation

Measuring stability across extended interaction sequences.

PSM proposes a shift from short-horizon evaluation to temporal coherence analysis.

Position

Language models generate text.

Persistent Self Models generate selves.

PSM does not replace foundation models.

It introduces a structural layer for self-continuity.
